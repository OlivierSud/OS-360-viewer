# Virtual Tour Editor — Fonctionnalités futures (vision produit)

---

# 1. Objectif du document

Ce document liste les fonctionnalités envisagées pour l’évolution du Virtual Tour Editor au-delà de la V3.

Ces fonctionnalités ne doivent **pas être implémentées en V1 ou V2**, mais servent de guide stratégique pour l’évolution du produit.

---

# 2. Vision globale

Le Virtual Tour Editor est destiné à évoluer vers une plateforme complète de création immersive :

- visites virtuelles 360°
- expérience interactive
- collaboration temps réel
- IA assistée
- publication SaaS

---

# 3. IA et automatisation

## 3.1 Génération automatique de liens

Le système pourra analyser les scènes et proposer automatiquement :

- connexions logiques entre pièces
- parcours optimisé
- suggestions de navigation

---

## 3.2 Positionnement automatique sur carte

L’IA pourra :

- analyser les panoramas
- détecter les pièces
- proposer une disposition spatiale cohérente sur la carte

---

## 3.3 Génération de hotspots intelligents

- détection d’objets dans la scène
- génération automatique de points d’intérêt
- descriptions contextuelles

---

## 3.4 Assistant d’édition

Un assistant intégré pourra :

- guider la création de visite
- proposer des améliorations UX
- corriger les incohérences de parcours

---
